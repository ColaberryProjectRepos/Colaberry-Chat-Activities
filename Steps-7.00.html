Step Name: Loading Data.

Steps: 1

      A. Click on Get Data

B.   Select SQL Server 

C.   Click Connect


Steps: 2

        Add Server name (To get the dataset credential,

       Download the doc  and click ok

       Select Database and table

        Select CCPP

        Select Tables to be imported: 

                                 - ADF_GroupMembers

                                 - ADF_GroupQuestionComments

                                 - ADF_GroupQuestions

                                 - ADF_Groups

       Click on Load-to-Load selected tables

                                Select Table
